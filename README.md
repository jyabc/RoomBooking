# RoomBooking
1. Clone this project
2. npm install

**Running on Android emulator**
> npx react-native run-android
<br/>

**Running on iOS emulator**
1. cd ios
2. pod install 
3. cd ..
> npx react-native run-ios
4. If fail, open XCode > roombooking.xcworkspace > File > Workspace Settings > change Build System to "Legacy Build System"

<br/>
To run on real device: https://reactnative.dev/docs/running-on-device

To run test:
> npm run test

<br/>

**Note:**
- Tested on real device for android, not tested on iPhone since I don't have apple developer id. <br/>
